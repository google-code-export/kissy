tab的触点为自适应宽度
注意：如果一级tab的触点的 ( fontSize + ( fontWeight == 'bold' ) ? 1 : 0 ) * 文字数 得到的结果为奇数的话，在ie6下的右圆角会出现1px的位置偏差。
要解决上述问题，你可以将触点文字设为偶数个，也可以将触点的宽度设置为偶数定宽。
面板容器如果要浮动，请设置宽度。
