kissy-ua.js

通过 KISSY.UA 的属性，你可以获取浏览器等用户代理的信息。属性值遵循以下规则：

  1. 表示当前引擎或浏览器的版本。版本号 1.2.3.4 会转换为数值 1.234
  2. 如果不是当前引擎或浏览器，返回 0

特别注意：

  1. KISSY.UA.mobile 返回的是字符串，目前支持 Apple, Nokia, Android 等设备的探测。
  2. KISSY.UA.flash 需要加载 swf-ua 模块才有值，否则始终为 0.

Properties

webkit - Number

webkit 的版本号。

chrome - Number

chrome 的版本号。

safari - Number

safari 的版本号。

gecko - Number

gecko 的版本号。

firefox - Number

firefox 的版本号。

ie - Number

ie 的版本号。

opera - Number

opera 的版本号。

mobile - String

mobile 的标志符。

flash - Number

flash 的版本号。

需要加载 swf-ua 模块才有值，否则始终为 0
