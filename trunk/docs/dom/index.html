这是 KISSY 的 DOM 模块，包含以下子模块：

  • selector CSS 选择器
  • dom query, get 基本操作
  • dom-traversal children, next, prev, parent 等遍历操作
  • dom-manipulation create, append, clone 等操作
  • dom-attr attr, removeAttr, text, val 属性操作
  • dom-class hasClass, addClass, removeClass, repaceClass, toggleClass 操作
  • dom-style css, addStyleSheet 操作
  • dom-region 获取位置、大小等信息
