这是 KISSY 的事件模块，包含以下子模块：

  1. event: 提供事件的添加、删除基本功能
  2. event-object: 封装 event 对象，提供一致的属性接口
  3. event-target: 自定义事件
  4. event-mouseenter: 让非 ie 浏览器支持 mouseenter/mouseleave
