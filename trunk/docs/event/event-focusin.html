event-focusin.js

让非 IE 浏览器也支持 focusin/focusout 事件，以实现 focus 的事件代理应用。

Examples

使用方式很简单，把 focusin/focusout 当成原生支持的事件一样使用就可：


var S = KISSY, Event = S.Event;

Event.on('#id', 'focusin focusout', function(ev) {
    // do something
});

希望 KISSY Event 能帮助你完成大部分事件操作代码。有任何建议，欢迎反馈给我们。
