event-mouseenter.js

mouseenter/mouseleave 非常有用，但目前只有 IE 支持。jQuery, YUI 等类库，都已内置了对这两个事件的支持。KISSY 也不例外，de-facto 才是真正的标准。

Examples

使用方式很简单，把 mouseenter/mouseleave 当成原生支持的事件一样使用就可：


var S = KISSY, Event = S.Event;

Event.on('#id', 'mouseenter', function(ev) {
    // do something
});

Event.on('#id', 'mouseleave', function(ev) {
    // do something
});
