什么是 alternative content

替换内容(alternative content)。

如果Flash插件没有安装或者不被支持(版本过低)那么这些替换内容就会被显示出来。

OBJECT tag only。参考 OBJECT fallback

示例： 

< object  type="application/x-shockwave-flash"  
data="http://img01.taobaocdn.com/tps/i1/T1S1tkXgliy0NtxVjX.swf"  
width="950" height="203" id="myFlashContent">
	< param name="movie" value="PATH2SWF.swf" / >
	< a href="http://www.adobe.com/go/getflashplayer" >
		< img src="get_flash_player.gif" alt="Get Adobe Flash player" / >
	< / a >
< / object>

FAQ

当Flash 对象(SWF)请求为404时是否被启用？
否，只有Flashplayer版本过低或未安装调用。
fallback在非IE下内容是否会被请求？
是，部分浏览器会请求此处内容，无论是否被满足条件。
